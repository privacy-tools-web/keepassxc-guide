# KeePassXC: The Password Manager That Never Talks to a Server

A lightweight, static guide to setting up and using KeePassXC — the open-source, offline password manager. No dependencies, no tracking, no server — everything runs directly in your browser.

## Contents

- **Download and install** — Windows, macOS, and Linux
- **Create your password database** — choosing a master password and key file
- **Adding and managing passwords** — entries, groups, and the password generator
- **Browser integration** — auto-fill in Firefox and Chrome via the KeePassXC-Browser extension
- **Importing existing passwords** — from Chrome, Firefox, Bitwarden, and others
- **Accessing passwords on your phone** — KeePassDX (Android) and Strongbox (iOS)
- **Backup strategy** — how to keep your database safe
- **FAQ & final checklist**

## Privacy

This is a fully static HTML page. No data is collected, transmitted, or stored. No external scripts, no analytics, no cookies of any kind.

## Usage

Download `keepassxc-guide.html` and open it in any modern browser. No build step, no install, no server required.

```bash
git clone https://github.com/privacy-tools-web/keepassxc-guide.git
open keepassxc-guide/keepassxc-guide.html
```
